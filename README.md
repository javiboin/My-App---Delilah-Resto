# My App - Delilah Resto

## Install Libraries:
```
npm i express -S
npm i moment -S
npm i swagger-ui-express swagger-jsdoc
```
## Start Server:
```bash
npm start 
```

or go to 

```bash
cd backend
node index.js
```

## Swagger:
```
http://localhost:3001/api-docs
```
