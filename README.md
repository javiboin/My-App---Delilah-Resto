# My App - Delilah Resto

## Install Libraries:
```
npm i 
```
## Install Database:
```
import file db/dellilah.db in MySQL Databases Location (C:\xampp\mysql\data) 
```
## Start Server:

```bash
npm run start
node src/index.js
nodemon src/index.js
```

## View Redis:
```Redis
npm run redis
```

## View Mocha:
```MochaJs
npm run rest
```
